O que é o Project Stok?

Esse é um protótipo de um site de gerenciamento de estoque desenvolvido por alunos de OO.

Essa plataforma foi feita com o intenção de:

1. O usuário possa ver as lojas cadastradas

2. O usuário possa ver os produtos cadastrados
